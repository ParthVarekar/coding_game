Challenge 2 (Boss): Sentinel Drone
ID: challenge_boss_sentinel
Task: Use a while loop to surge power ONLY when shields are down.

FULL SOLUTION:
# We must define the command first
def surge_command():
    print('SURGING POWER')
    return True

# The sequence of shield fluctuations
shield_states = ['up', 'up', 'up', 'down', 'up']

# Loop through the states until we find 'down'
while len(shield_states) > 0:
    current = shield_states.pop(0)
    if current == 'down':
        surge_command()
        break

import io

memory_csv = "timestamp,load,stability\n101,0.45,0.9\n102,,0.85\n103,0.72,"
df = pd.read_csv(io.StringIO(memory_csv))

# Step 1 & 2: Fill missing values
df['load'] = df['load'].fillna(df['load'].mean())
df['stability'] = df['stability'].ffill()

# Step 3: Filter
diana_memory = df[df['stability'] > 0.8]
